# 🏠 House Price Prediction in India

## Overview

This project aims to predict residential property prices across Indian cities using Machine Learning techniques. The project covers the complete data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, model building, and model evaluation.

---

## Objectives

- Clean and preprocess raw housing data
- Perform exploratory data analysis (EDA)
- Engineer meaningful features
- Build and compare multiple machine learning models
- Predict house prices with high accuracy

---

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost (planned)
